## Building Atomlang From Source:

- this guide if to build atomlang from source

- Linux:

```bash
sudo apt-get update
sudo apt install gcc make
git clone https://github.com/atomlang/atomlang
make
```

- macOS:

```bash
brew update
brew install gcc make
git clone https://github.com/atomlang/atomlang
make
```

- Windows:

```bash
git clone https://github.com/atomlang/atomlang
cmd /c build.bat
```
